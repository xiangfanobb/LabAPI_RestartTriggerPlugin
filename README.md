# RestartTriggerPlugin - 远程重启触发器
# 插件概述
RestartTriggerPlugin 是一个基于 HTTP 协议的远程重启触发插件，允许授权用户通过简单的 HTTP 请求安全地触发服务器重启操作。插件采用鉴权机制和冷却时间控制，确保重启操作的安全性和可控性。

# 功能特性
# 🔐 安全特性
密钥鉴权：必须提供正确的密钥才能触发重启

冷却时间控制：防止频繁触发重启（默认300秒）

GET请求限制：只接受 GET 请求，简化调用方式

详细日志记录：所有操作均有详细日志记录

# 🌐 网络配置
灵活端口：可自定义监听端口（默认8080）

跨域兼容：支持从任何客户端调用

异步处理：请求立即响应，重启异步执行

# ⚙️ 配置管理
自动配置生成：首次运行自动生成配置文件

YAML格式配置：使用人类可读的 YAML 格式

配置验证：端口范围、密钥非空等自动验证

热重载支持：修改配置后重启插件生效

# 技术规格
依赖项
框架：LabAPI

序列化：YamlDotNet

网络：System.Net.HttpListener

# 系统要求
.NET 兼容环境

LabAPI 框架支持

系统防火墙开放指定端口

# 安装部署
1. 安装插件
将插件 DLL 文件放入 LabAPI 的插件目录中，框架启动时会自动加载。
